
# GenomeMethyl: DNA Methylation Prediction Using Machine Learning

## Overview

GenomeMethyl is a machine learning project that predicts DNA methylation status based on genomic features. The project explores how characteristics such as CpG density, genomic location, regulatory score, and conservation score can be used to classify methylated and unmethylated genomic regions.

## Dataset

* Records: 1,000
* Features:

  * cpg_density
  * genomic_location
  * regulatory_score
  * conservation_score
* Target:

  * methylation_status

## Project Workflow

1. Exploratory Data Analysis (EDA)
2. Data Preprocessing
3. Model Training
4. Model Evaluation
5. Feature Importance Analysis

## Models Used

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier
